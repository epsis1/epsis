01 — About

What is EPSIS?

A complete indoor environment monitoring system built from scratch on a single ESP32 chip — no cloud subscription, no monthly fees.

This project builds a low-cost indoor environmental monitoring system using sensors including DHT11, BH1750, MQ-135, MQ-2, Sound sensor and dual IR sensors with an ESP32 microcontroller. It continuously tracks air quality, temperature, humidity, gas leaks, light intensity, sound levels and room occupancy. Data is displayed locally on an OLED screen and remotely via a WiFi web dashboard. The system triggers smart tiered alerts through a buzzer and LED — with different tones for different hazards. All readings are logged to onboard memory every 60 seconds, the person counter survives power cuts, and the dashboard is password protected. Designed with future AI-based enhancements in mind.
